## 📱 App Overview

**CUBS Visa Management** is a comprehensive employee management system designed for companies to manage visa documentation, employee records, and compliance monitoring.

### Key Features to Test

#### 1. Employee Management
- **Location**: Dashboard → Employees
- **Test**: Add, edit, and view employee records
- **Demo Data**: Pre-populated with sample employees

#### 2. Document Management
- **Location**: Dashboard → Documents
- **Test**: Upload, view, and manage employee documents
- **Features**: PDF preview, bulk operations, search functionality

#### 3. Visa Monitoring
- **Location**: Dashboard → Notifications
- **Test**: View visa expiry alerts and notifications
- **Features**: Real-time monitoring, email notifications

#### 4. Admin Dashboard
- **Location**: Dashboard (main screen)
- **Test**: View analytics, employee statistics, recent activities
- **Features**: Charts, graphs, quick actions

#### 5. User Management
- **Location**: Admin → Users
- **Test**: Manage user accounts and permissions
- **Features**: Role-based access control
