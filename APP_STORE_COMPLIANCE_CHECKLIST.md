## 📱 APP STORE SPECIFIC REQUIREMENTS

### Google Play Store
- **Target API Level**: API 34 (Android 14) minimum
- **Minimum SDK**: API 22 (Android 5.1) recommended
- **AAB Format**: Required for new apps (replaces APK)
- **Content Rating**: Required for all apps
- **Privacy Policy**: Required URL in store listing
- **Target Audience**: Age rating required

### Apple App Store
- **Minimum iOS Version**: iOS 13.0 recommended
- **Target iOS Version**: iOS 17.0 or latest
- **IPA Format**: Required for App Store submission
- **App Review**: Manual review process (usually 24-48 hours)
- **Privacy Nutrition Labels**: Required for apps with data collection
- **Export Compliance**: Required for certain app types
- **Guideline 2.3.10**: Web content must be more than just a website
